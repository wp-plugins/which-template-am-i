== Description ==

This plugin will display the name of the WordPress Template in use for the current page at the bottom of the page. It will only show in the front end when you are logged in.

== Installation ==

This section describes how to install the plugin:

1. Upload the `which-template-am-i` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==
Just activate the plugin and from the front end view of a WordPress theme, look at the bottom of the page to see which WordPress template the current theme is using.
